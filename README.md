# BalmJS template - `react-master`

> A React project with router & MobX

## Installation

```sh
npm install -g balm-cli
balm init react-master my-project

cd my-project
yarn
# or
npm i
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

- thx [BalmJS](https://balmjs.com/)
